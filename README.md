# PixelWars

PixelWars는 플레이어와 설정 가능한 수의 AI가 전쟁하는 파이썬/Pygame 기반 전략 게임입니다.

## 실행 방법

```powershell
cd C:\Coding\파이썬\PixelWars
py -m pip install -r requirements.txt
py main.py
```

Windows에서는 `run.bat`를 실행해도 됩니다.
현재 게임 버전은 `v1.0.1-alpha`입니다.
저장 파일은 소스 실행에서는 `saves/*.pxw`, EXE 실행에서는 `%LOCALAPPDATA%\PixelWars\saves\*.pxw`로 생성됩니다. `.pxw`는 PixelWars 저장 파일 형식입니다.
세이브에는 맵, 영토, 돈, 병력, 세력 성향, 건물, 유닛, 작전, 전쟁 상태, 핵 낙진 지역이 저장됩니다.

## 설치 파일

GitHub Release의 `PixelWars.exe`는 바로 실행 가능한 게임 파일입니다.
`PixelWarsInstaller.exe`는 최신 릴리즈를 확인하는 설치/업데이트 파일입니다.

- 설치 위치: `%LOCALAPPDATA%\PixelWars`
- 설치 버전 기록: `%LOCALAPPDATA%\PixelWars\version.txt`
- 현재 설치가 없거나 최신 버전보다 낮으면 기존 설치 폴더를 삭제하고 다시 설치합니다.
- 설치 후 바탕화면 바로가기를 만들고 시작프로그램에 `PixelWars.lnk`를 등록합니다.
- 재설치 전 실행 중인 `PixelWars.exe`를 종료합니다.
- 설치가 끝나면 게임을 자동 실행합니다.

## 로비와 매칭

게임을 실행하면 바로 전투가 시작되지 않고 로비가 먼저 열립니다.

- 로비에서 선택된 맵, 맵 크기, 전쟁 규모, 전략 템포, 승리 조건을 확인할 수 있습니다.
- `매칭 시작`을 누르거나 `Enter`/`Space`를 누르면 로컬 매칭이 시작됩니다.
- 매칭은 플레이어 1명과 선택한 전쟁 규모의 AI 수를 채운 뒤 수도 선택 단계로 넘어갑니다.
- 매칭 중 `ESC`를 누르면 로비로 돌아갑니다.
- 로비에서 `1`, `2`로 맵을 바꾸고, `R`로 랜덤 맵을 다시 뽑을 수 있습니다.
- 로비에서 `3` 또는 `크기` 버튼으로 맵 크기를 `작음 / 보통 / 큼 / 초거대` 중 선택할 수 있습니다.
- 로비에서 `4` 또는 전쟁 규모 버튼으로 `소규모 전쟁 / 표준 전쟁 / 대규모 전쟁 / 세계 대전 / 혼돈` 중 선택할 수 있습니다.
- 로비에서 `5` 또는 `템포` 버튼으로 `표준 / 장기전 / 강한 AI` 밸런스를 선택할 수 있습니다.
- 저장 파일은 로비의 `저장 불러오기` 버튼이나 게임 중 `F9`로 불러올 수 있습니다.
- 수도 선택 이후 진행 중인 게임은 60초마다 `autosave_*.pxw`로 자동저장되며, 최근 5개만 유지합니다.

### 온라인 로비

현재 온라인 시스템은 같은 PC 또는 같은 LAN에서 테스트할 수 있는 가벼운 TCP 로비입니다.

- `온라인 방 만들기`: 내 PC에서 로비 서버를 열고 Host로 참가합니다.
- `로컬 참가`: `127.0.0.1:50505` 로비에 참가합니다. 같은 PC 테스트용입니다.
- 다른 PC에서 참가하려면 로비의 IP 입력칸에 방장의 `방 주소` IP를 입력한 뒤 `IP 참가`를 누릅니다.
- 포트는 `50505`를 사용합니다.
- 별도 서버 PC에서는 `py -m pixelwars.online`으로 로비 서버만 실행할 수 있습니다.
- 현재 단계에서는 온라인 참가자 목록과 매칭 흐름을 동기화합니다.
- 온라인 로비에서는 참가자별 `READY` 상태를 표시하고, `준비` 버튼으로 내 준비 상태를 바꿀 수 있습니다.
- 실제 전투 명령 동기화는 다음 단계에서 서버 권한 방식으로 확장할 예정입니다.

## 포함된 맵

맵 파일은 `maps` 폴더에 들어 있습니다. 게임은 `maps/*.png`를 자동으로 읽습니다.
랙을 줄이기 위해 로비에서 맵 전투 해상도를 선택할 수 있습니다. `작음`은 최대 한 변 260픽셀, `보통`은 360픽셀, `큼`은 460픽셀, `초거대`는 보통 맵의 약 8배 면적인 최대 한 변 1018픽셀로 리샘플링합니다.

- 기본: `worldMap.png`
- 커스텀맵 5종: `custom_archipelago.png`, `custom_twin_seas.png`, `custom_ringlands.png`, `custom_shattered_coast.png`, `custom_crossroads.png`, `continent_aurora.png`, `continent_borealis.png`, `continent_crescent.png`, `continent_delta.png`, `continent_equinox.png`
- 대륙맵 4종: `asia.png`, `Europe.png`, `Africa.png`, `America.png`

새 게임은 `maps/*.png` 중 하나를 랜덤으로 시작합니다.
게임 안에서 `1`, `2` 키로 이전/다음 맵을 바꾸고, `R` 키로 랜덤 맵을 다시 뽑을 수 있습니다.

## 기본 규칙

- 한 맵에는 플레이어 1명과 로비에서 선택한 전쟁 규모만큼 AI 세력이 생성됩니다.
- 전쟁 규모는 소규모 전쟁 20명, 표준 전쟁 40명, 대규모 전쟁 60명, 세계 대전 80명, 혼돈 100명입니다.
- `장기전` 템포는 생산과 AI 진격 간격을 늦춰 전선이 오래 유지되게 합니다.
- `강한 AI` 템포는 AI 시작 자원과 행동 확률을 올려 더 거친 판을 만듭니다.
- AI는 맵의 육지 픽셀 일부를 시작 영토로 받습니다.
- 플레이어는 게임 시작 시 직접 육지를 좌클릭해 시작지점, 즉 수도를 정합니다.
- 수도를 정하면 주변 픽셀이 플레이어 시작 영토가 되고, 도시 1개와 공장 1개만 생성됩니다.
- 영토는 픽셀 단위로 계산됩니다.
- 맵 색상은 흰색 계열을 육지, 하늘색 계열을 바다로 판정합니다.
- 지형은 평원/산맥/사막으로 나뉩니다. 평원은 일반 지형이고, 산맥과 사막은 진격 속도가 20% 느립니다.
- 영토는 더 이상 공장 거리 때문에 붕괴되지 않습니다.
- 대신 진격에는 보급이 필요합니다. 공장은 주변 8칸만 직접 보급하고, 공장에 연결된 보급로는 주변 36칸까지 진격을 허용합니다.
- 보급로는 공장 또는 활성 보급로와 보급 연결 거리 안에 있으면 작동합니다.
- 작동 중인 보급 범위는 초록색 테두리 원으로 표시되고, 연결된 보급로는 초록색 선으로 이어집니다.
- 공장과 연결된 보급로가 전부 사라져 보급망이 끊기면, 그 시점부터 3분 동안 병력이 최대 75%까지 약화됩니다.
- 보급망을 다시 연결하면 병력 약화 카운트다운은 중단됩니다.
- 공격은 상하좌우 `+` 방향으로만 확산됩니다.
- 좌클릭하면 클릭한 땅의 주인과 슬라이더로 선택한 병력이 하나의 `작전`으로 묶입니다.
- 주인이 없는 땅은 `빈땅(None)` 개척 작전이 됩니다.
- 작전은 자기 영토와 닿아 있는 대상 영토 또는 빈땅을 향해 자동으로 전진합니다.
- 진행 중인 작전은 지도 위 화살표로 공격 방향을 표시합니다. 화살표는 가능한 육지 진격 경로를 따라가며, 플레이어 작전은 파란색, AI 작전은 붉은색 계열로 보입니다.
- 작전 병력은 픽셀 공격마다 줄어들고, 병력을 다 쓰면 해당 작전은 자동 파기됩니다.
- 작전이 끝났을 때 병력이 남아 있으면 국가 총 병력으로 복귀합니다.
- `E` 키를 누르면 작전 화살표와 사이드바 작전 목록을 확인하고 `취소` 버튼으로 중단할 수 있습니다.
- 작전을 취소하면 남은 작전 병력은 국가 총 병력으로 복귀합니다.
- 기본 시작 병력은 `1000`입니다.
- 빈땅 점령은 병력을 소모하지 않습니다.
- 상대 병력이 거의 없다면 병력을 소모하지 않고 점령할 수 있습니다.
- 적 영토 점령 소모량은 고정 픽셀 비용이 아니라 현재 상대 예비 병력과 병력 밀도에 비례합니다.
- 세력별 병력 최대치는 `점령 픽셀 수 × 픽셀당 최대 병력`입니다. 좁은 영토에서는 병력을 무한히 쌓을 수 없습니다.
- 병력으로 상대 건물이 있는 픽셀을 점령하면 건물을 삭제하지 않고 소유권을 가져옵니다.
- 미사일에 맞은 건물은 기존처럼 파괴됩니다.
- 탄도미사일은 폭격기 없이 단독으로 날아가지만 공장 또는 공군기지가 있어야 발사할 수 있고, 방공미사일기지 근처 목표에서는 65% 확률로 요격됩니다.
- 방공미사일기지가 여러 개여도 탄도미사일 요격 확률은 65%로 고정됩니다.
- 탄도미사일은 45초, 핵탄도미사일은 90초 재장전 시간이 있습니다.
- 핵탄도미사일은 폭발하면 반경 30칸의 육지와 건물을 빈땅으로 만들고, 10분 동안 해당 지역을 낙진 지역으로 남깁니다.
- 낙진 지역은 다시 점령해도 주기적으로 주인이 사라져 빈땅이 됩니다.
- 핵탄도미사일에 의해 오염된 낙진 지역은 승리 판정의 점령 가능 영토에서 제외됩니다.
- `None`이 아닌 플레이어/AI를 공격하면 전쟁 상태에 등록됩니다.
- 현재 전쟁 상태는 둘 중 한 세력이 멸망할 때까지 유지됩니다. 추후 평화협정과 동맹 기능을 추가할 예정입니다.
- 오염 지역을 제외한 점령 가능 영토의 95% 이상을 점령하고, 모든 적이 5픽셀 이하를 보유한 상태를 2분 유지하면 승리합니다.
- 플레이어가 가진 픽셀이 0이 되면 즉시 패배합니다.
- 전선 업데이트는 60FPS 루프에서 매 프레임 처리됩니다.
- 좌클릭 작전에 투입할 총 병력은 오른쪽 패널의 슬라이더로 조절합니다. 슬라이더는 `0%~100%` 비율 기준이며, 실제 투입 병력은 `현재 보유 병력 × 선택 비율`입니다.
- 건물 건설과 도시 레벨업에는 5초 쿨타임이 있습니다.

## AI 성향

AI마다 성향이 하나씩 배정됩니다.

- `균형`: 공격, 건설, 공습을 적당히 사용합니다.
- `공격적`: 더 자주 작전을 만들고 더 많은 병력을 투입합니다.
- `방어`: 공격은 적게 하고 방공미사일기지와 공장을 자주 짓습니다.
- `경제`: 도시와 공장을 선호해서 성장 위주로 움직입니다.
- `약탈`: 빠른 공격과 제한적인 공습/탄도미사일을 사용합니다.
- `극공`: 매우 자주 공격하고 공습도 쓰지만 방어 건설은 거의 하지 않습니다.
- `극방`: 거의 공격하지 않고 방공/공장/도시 위주로 요새화합니다.

세력 순위 패널에는 AI 이름 옆에 성향이 함께 표시됩니다.
표준 템포의 AI는 플레이어와 같은 시작 자원으로 시작하고, 인접 적의 영토와 병력 규모를 보고 공격 대상을 고릅니다.
주변에 적이 없으면 낮은 확률로 빈땅 개척 작전을 만들며, 보급로는 전선 가까이에 깔려고 합니다.
공습과 탄도미사일은 공장, 보급로, 공군기지, 방공기지 같은 전략 건물을 우선 목표로 삼지만 사용 빈도는 낮게 제한됩니다.

## 조작

- `H`, `F1`, `?`: 인게임 도움말 열기/닫기
- 로비 `Enter`/`Space`: 매칭 시작
- 매칭 중 `ESC`: 매칭 취소 후 로비 복귀
- 로비 `3`: 맵 크기 변경
- 로비 `4`: 전쟁 규모 변경
- 로비 `5`: 전략 템포 변경
- 로비 버튼 `온라인 방 만들기`: LAN 로비 서버 시작
- 로비 버튼 `로컬 참가`: 같은 PC의 온라인 로비 참가
- 로비 IP 입력칸 + `IP 참가`: 같은 LAN의 다른 PC 로비 참가
- 게임 시작 전 `좌클릭`: 플레이어 수도를 지정합니다.
- 수도 선택 중 `C`: 수도 자동 배치
- 게임 시작 후 `좌클릭`: 클릭한 픽셀의 주인과 슬라이더 병력을 묶어 작전을 생성합니다.
- `우클릭`: 작전 메뉴를 엽니다.
- `WASD`: 맵 이동
- `Shift + WASD`: 빠른 맵 이동
- `마우스 휠`: 맵 확대/축소
- `ESC`: 도움말/열린 메뉴 닫기, 게임 중 일시정지 메뉴 열기/계속하기
- `Tab`: 사이드바 접기/펼치기
- 사이드바 위 `마우스 휠`: 사이드바 스크롤
- 사이드바 미니맵 클릭: 해당 위치로 카메라 이동
- `C`: AI 성향과 인접 AI 위협도 표시 켜기/끄기
- `V`: 보급 연결선/범위와 보급 추정 표시 켜기/끄기
- `E`: 작전 화살표/작전 목록/전쟁 상태 표시 켜기/끄기
- `G`: 건물 표시 켜기/끄기
- 확대가 3x 이상이면 건물 표시는 자동으로 숨겨집니다.
- `T`: 영토 이미지 표시 켜기/끄기
- `F5`: 현재 게임을 `.pxw` 파일로 저장
- `F9`: `.pxw` 저장 파일 선택 창 열기. 선택 창에는 맵, 전쟁 규모, 맵 크기, 템포, 병력 요약이 표시됩니다.
- `L`: 현재 게임을 버리고 로비로 돌아가기
- `-`, `=`: 게임 속도 0.5x/1x/2x/4x 조절. 온라인 참가자가 1명이라도 있으면 1x로 고정됩니다.
- `F`: 전투기 구매
- `B`: 폭격기 구매
- `N`: 전투함 구매
- `1`, `2`: 이전/다음 맵 변경
- `R`: 랜덤 맵으로 새 게임 시작

게임을 처음 실행하면 도움말이 자동으로 표시됩니다. 도움말은 클릭해도 닫힙니다.
사이드바의 세계 로그에는 나라 멸망과 핵 발사 같은 주요 사건이 최근 순서로 표시됩니다.

## 영토 이미지

`territory_images/` 폴더에 이미지를 넣거나 일시정지 메뉴의 설정에서 업로드하면 플레이어 영토 위에 이미지가 입혀집니다.

- `T` 키 또는 일시정지 메뉴의 설정에서 영토 이미지 표시를 켜고 끌 수 있습니다.
- 설정에서 업로드, 이전/다음 이미지 선택, 채우기 방식 변경을 할 수 있습니다.
- 채우기 방식은 타일, 늘려 채우기, 잘라서 채우기를 지원합니다.
- `player.png`, `0.png`, `faction_0.png`가 있으면 우선 선택하며, 그 외 PNG/JPG/JPEG/BMP 파일도 선택할 수 있습니다.
- 현재는 플레이어 영토 이미지부터 지원합니다.
- 세이브 파일에는 선택한 이미지 이름과 채우기 방식도 저장됩니다.

## 일시정지 메뉴

`ESC` 키를 누르면 일시정지 메뉴가 열립니다.

- 계속하기
- 설정
- 저장
- 불러오기
- 게임 나가기
- 게임 종료

설정 안에서는 건물 표시, 보급 레이어, 영토 테두리, 영토 이미지 표시, 영토 이미지 선택, 채우기 방식, 이미지 업로드를 바꿀 수 있습니다.

## 우클릭 작전 메뉴

### 상륙공격

우클릭한 땅의 주인을 타겟으로 지정하고, 슬라이더로 선택한 병력을 실은 상륙선을 플레이어 영토 중 가장 가까운 해안 픽셀에서 출발시킵니다.

- 목표 지점이 해변이 아니면 가장 가까운 해변으로 자동 보정됩니다.
- 플레이어 영토에 해안이 없으면, 플레이어 영토에서 가장 가까운 해안을 임시 출항 지점으로 사용합니다.
- 상륙선은 육지를 직선으로 뚫지 않고 바다 픽셀 경로를 따라 이동합니다.
- 상륙선은 매우 느리게 이동합니다.
- 바다 경로는 육지를 밟지 않는 선에서 자동 압축되어 너무 오래 걸리지 않게 이동합니다.
- 연결된 바닷길이 없으면 상륙공격은 실패합니다.
- 상륙선이 도착하면 목표 해변 픽셀 1칸을 점령하고 `상륙 작전`이 생성됩니다.
- 상륙 작전은 일반 작전과 다르게 전체 아군 전선에서 퍼지지 않습니다.
- 상륙 작전은 처음 점령한 해변 픽셀과, 그 작전이 새로 점령한 픽셀에서만 뻗어나갑니다.
- 상륙 작전은 시작 후 1분 동안 보급망 밖에서도 유지/진격할 수 있고, 그 뒤에는 보급망이 필요합니다.
- 병력을 다 쓰면 상륙 작전도 자동 파기됩니다.

### 공중공격: 전투기

공군기지에서 전투기를 출격시켜 우클릭한 땅으로 보냅니다.

- 공군기지가 필요합니다.
- 전투기를 미리 구매해야 합니다.
- 도착하면 해당 픽셀을 점령하고 다시 공군기지로 돌아갑니다.
- 전투기 강습 작전은 시작 후 1분 동안 보급망 밖에서도 유지/진격할 수 있고, 그 뒤에는 보급망이 필요합니다.
- 방공미사일기지 근처에서는 격추될 수 있습니다.

### 공중공격: 폭격기

공군기지에서 폭격기를 출격시켜 우클릭한 땅에 미사일을 투하합니다.

- 공군기지가 필요합니다.
- 폭격기를 미리 구매해야 합니다.
- 미사일은 주변 건물을 파괴하고 땅을 빈땅으로 만듭니다.
- 투하 후 폭격기는 공군기지로 돌아갑니다.
- 방공미사일기지 근처에서는 격추될 수 있습니다.
- AI는 돈이 충분하고 공군기지가 있을 때 제한적으로 미사일성 폭격기를 보내기도 합니다.

### 탄도미사일

폭격기 없이 단독으로 목표 지점까지 날아가는 고가 미사일입니다.

- 돈이 필요하며, 아군 공장 또는 공군기지에서만 출발합니다.
- 목표 근처에 적 방공미사일기지가 있으면 65% 확률로 요격됩니다.
- 방공미사일기지가 여러 개 있어도 요격 확률은 65%로 고정됩니다.
- 명중하면 중심 픽셀을 빈땅으로 만들고, 인접한 건물을 파괴합니다.
- AI도 공군기지가 있고 돈이 충분할 때 아주 낮은 확률로 사용합니다.

### 핵탄도미사일

탄도미사일의 대형 버전입니다.

- 폭발하면 반경 30칸의 육지 영토와 건물이 사라집니다.
- 폭발 지역은 10분 동안 붉은 낙진 지역으로 표시됩니다.
- 낙진 지역은 다시 점령하더라도 주기적으로 빈땅으로 돌아갑니다.
- 요격 규칙은 탄도미사일과 동일하게 65% 고정입니다.

### 전투함 배치

전투함은 바다 위에 배치됩니다.

- 전투함 구매에는 항구가 필요합니다.
- 우클릭한 곳이 육지면 가장 가까운 바다로 자동 보정됩니다.
- 근처에 적 상륙선이 오면 요격용으로 대기합니다.

## 건설 메뉴

자신의 땅을 우클릭하면 `건설` 메뉴가 추가로 나타납니다.

### 공군기지

전투기와 폭격기를 구매하고 출격시키기 위한 건물입니다.

### 공장

보급망의 시작점입니다.

- 공장 자체도 보급원으로 작동합니다.
- 공장 단독 보급 범위는 8칸으로 짧습니다.
- 보급로는 공장과 보급 연결 거리 안에 있어야 작동합니다.
- 긴 전선은 보급로를 이어서 확장해야 합니다.

### 보급로

공장 보급 범위를 확장하는 건물입니다.

- 자신의 땅에 건설할 수 있습니다.
- 공장 또는 활성 보급로와 보급 연결 거리 안에 있어야 작동합니다.
- 작동 중인 보급로 주변 36칸 안에서는 작전이 계속 전진합니다.
- 플레이어의 작동 중인 보급 범위는 맵 위에 초록색 테두리 원으로 표시됩니다.
- 연결된 보급로는 초록색 선 형태로 표시됩니다.

건물 건설과 도시 레벨업은 한 번 실행한 뒤 5초 쿨타임이 필요합니다.

### 항구

수입/수출용 건물입니다.

- 해변 픽셀에만 건설할 수 있습니다.
- 주기적으로 돈을 생성합니다.
- 전투함 구매 조건입니다.

### 도시

주기적으로 병력과 돈을 생산합니다.

- 최대 5레벨까지 레벨업할 수 있습니다.
- 레벨이 높을수록 생산량이 증가합니다.

### 방공미사일기지

근처로 오는 전투기와 폭격기를 80% 확률로 격추합니다.

- 격추에 실패하면 근처 땅에 미사일이 떨어질 수 있습니다.
- 미사일은 건물을 삭제하고 해당 지역을 빈땅으로 만듭니다.

## 밸런스 조정 위치

주요 수치는 `pixelwars/game.py` 안에서 조정할 수 있습니다.

- 초기 돈과 병력: `Faction`
- 로비 전쟁 규모/AI 수 선택지: `AI_COUNT_OPTIONS`, `AI_COUNT_LABELS`
- 로비 맵 크기 선택지: `MAP_SIZE_OPTIONS`
- 전략 템포/장기전/강한 AI 배율: `PACE_OPTIONS`
- 작전 투입 병력 비율 슬라이더: `operation_percent`, `selected_operation_troops`, `update_slider`
- 빈땅 점령 소모 병력: `PIXEL_ATTACK_COST`
- 무저항 판정 병력 기준: `NO_RESISTANCE_TROOPS`
- 적 병력 비례 소모량: `ENEMY_TROOP_COST_RATIO`, `MAX_ATTACK_COST`
- 적 병력 밀도 소모량: `ENEMY_DENSITY_COST_RATIO`
- 픽셀당 병력 보유 최대치: `TROOPS_PER_PIXEL_CAP`
- 공습/상륙 보급 유예 시간: `TEMP_OPERATION_GRACE`
- 보급 범위: `SUPPLY_FACTORY_RANGE`, `SUPPLY_RANGE`
- 탄도미사일 비용/속도/요격률: `BALLISTIC_COST`, `BALLISTIC_SPEED`, `BALLISTIC_INTERCEPT_CHANCE`
- 핵탄도미사일 비용/반경/낙진 시간: `NUKE_COST`, `NUKE_RADIUS`, `FALLOUT_DURATION`
- 미사일 재장전: `BALLISTIC_COOLDOWN`, `NUKE_COOLDOWN`
- 건설 쿨타임: `BUILD_COOLDOWN`
- 보급망 상실 후 병력 약화 시간/하한선: `SUPPLY_COLLAPSE_TIME`, `SUPPLY_COLLAPSE_FLOOR`
- 건설 비용: `build`
- 항공기/전투함 구매 비용: `buy_aircraft`, `buy_ship`
- 도시/항구 생산량: `update_economy`
- 방공 격추 확률: `air_defense_hits`
- 프레임당 공격 픽셀 수: `ATTACKS_PER_FRAME`
- AI 공격/판단 간격: `PACE_OPTIONS`의 `attack_interval`, `ai_decision_interval`
- AI 성향별 행동 확률: `AI_PERSONALITIES`
